# Настройка авторизации через Telegram

## Шаг 1: Создание Telegram бота

1. Откройте Telegram и найдите бота [@BotFather](https://t.me/BotFather)
2. Отправьте команду `/newbot`
3. Следуйте инструкциям и создайте бота
4. Сохраните полученный токен (например: `123456789:ABCdefGHIjklMNOpqrsTUVwxyz`)

## Шаг 2: Настройка переменных окружения

Откройте файл `backend/.env` и добавьте токен бота:

```env
TELEGRAM_BOT_TOKEN=ваш_токен_от_BotFather
JWT_SECRET=ваш_секретный_ключ_для_jwt
```

## Шаг 3: Запуск приложения

```bash
docker-compose up -d --build
```

## Шаг 4: Использование

1. Откройте ваш Telegram бот
2. Отправьте команду `/start`
3. Получите код авторизации
4. Откройте http://localhost:3003/auth
5. Введите ваш Telegram ID (можно узнать у бота @userinfobot)
6. Введите код авторизации
7. Нажмите "Войти"

## Как узнать свой Telegram ID

1. Найдите бота [@userinfobot](https://t.me/userinfobot) в Telegram
2. Отправьте команду `/start`
3. Бот вернет ваш Telegram ID

## Примечания

- Код авторизации действителен 10 минут
- После использования код удаляется
- JWT токен действителен 7 дней
- При следующем входе нужно снова получить новый код через `/start`

## Как использовать:

1. Откройте http://sybota.space/auth
2. Прокрутите вниз до блока "Тестовый режим"
3. Нажмите кнопку "Войти как тестовый аккаунт"
4. Вы автоматически войдете в систему

## Отключение тестового аккаунта:

Если хотите отключить тестовый аккаунт в production, установите в `docker-compose.yml`:

```yaml
ENABLE_DEV_LOGIN: false
```

Или через переменную окружения:

```bash
ENABLE_DEV_LOGIN=false docker-compose up -d --build
```

## Безопасность:

⚠️ **Важно:** Тестовый аккаунт позволяет войти без Telegram авторизации. Используйте его только для тестирования. В production рекомендуется отключить его после завершения тестов.
